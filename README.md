## 🚀 Overview

**Vortex** is not just another downloader — it's a professional-grade **media acquisition and organization engine** built for power users.

Powered by **yt-dlp** and wrapped in a modern, high-performance UI using **Flet**, Vortex delivers fast downloads, clean organization, preview tools, and advanced processing features while running on both **Windows and Android**.

---

## ✨ Key Features

| Feature | Description |
| :--- | :--- |
| 📂 **Smart Folders** | Automatically organizes downloads into folders based on **Channel** or **Playlist name**. |
| 🛡️ **SponsorBlock Support** | Auto-skip sponsored segments, intros, outros, and irrelevant sections for clean content. |
| ⚡ **Instant Preview** | Paste a link to preview **thumbnail, title, duration and uploader** before download. |
| ✂️ **Clip Precision** | Download only the part you need — cut by timestamps without saving the entire video. |
| 🎬 **MP4 & Subtitles** | Force MP4 format for full compatibility and optionally embed subtitles into videos. |
| 📋 **Clipboard Monitor** | Automatically detects video links copied to your clipboard. |
| 📦 **Playlist & Batch Mode** | Download entire playlists or process multiple URLs at once. |
| 📱 **Android 7 → 16 Support** | Fully working on modern Android devices, including scoped-storage handling. |
| 📁 **Public Downloads Folder** | Files are saved to:  
  `Download/Vortex` — fully accessible via any file manager. |

## 🛠️ Built With

* **Language:** Python `3.12+`
* **UI Framework:** Flet (Flutter-based Python GUI)
* **Core Downloader:** yt-dlp
* **Media Processing:** FFmpeg
* **Community APIs:** SponsorBlock
